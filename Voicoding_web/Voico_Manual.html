Assertion variables 사용 방법

변수 선언에는 몇 가지의 큰 틀이 존재합니다.

1. 변수 선언 문자


변수 선언 문자를 하는 것은 a = 'Hello World' 와 같은 경우를 의미합니다. [변수 선언 문자 a = Hello World] --> a = 'Hello World'와 같이 출력됩니다.

2. 변수 선언 숫자


변수 선언 숫자를 하는 것은 a = 4 와 같은 경우를 의미합니다. [변수 선언 숫자 a = 4] --> a = 4 와 같이 출력됩니다.

3. 변수 선언 리스트


( 변수 선언 리스트를 하는 것은 a = [], a = [1, 2, 3] 과 같은 경우를 의미합니다. [변수 선언 리스트 a = 빈리스트 , 변수 선언 리스트 a = 1, 2, 3] --> a = [], a = [1, 2, 3]와 같이 출력됩니다.

4. 변수 선언 함수


변수 선언 함수를 하는 것은 a=sum(), a=len() 과 같은 내장함수를 사용하거나 사용자 정의 함수를 사용하는 경우를 의미합니다. [변수 선언 함수 a=, 기본 함수 사용 sum, 파라미터 선언 c] --> a = sum(c) 과 같이 출력됩니다.

(파라미터 선언은 Assertion Parameters를 참고해주세요.)


Assertion Parameters 사용 방법

파라미터 선언은 대다수의 함수 선언, 내장함수 사용, 반복문 사용에 있어서 꼭 사용해야 하니 숙지해주세요.
파라미터 선언을 하는 방법은 다음과 같습니다.
만약, 파라미터로 a b c 를 선언하고 싶다면, [파라미터 선언 a, b, c]라고 읽어주시면 해당하는 곳에 파라미터로써 입력됩니다.

Basic Func 사용 방법

Basic Func에는 파이썬의 내장 함수들이 들어있습니다.
대체적으로 같은 틀을 가지고 있으니 방법을 숙지해주세요.

input의 경우

, [변수 선언 함수 a =, 기본 함수 사용 input] --> a = input(); 와 같이 출력됩니다.

print의 경우

, [기본 함수 사용 print, 파라미터 선언 a] --> print(a); 와 같이 출력됩니다.

sum, max, min, len의 경우

[기본 함수 사용 sum, 파라미터 선언 a] --> sum(a); 와 같이 출력됩니다.

(파라미터 선언은 Assertion Parameters를 참고해주세요.)


(변수 선언은 Assertion Variables를 참고해주세요.)

Loop statements 사용 방법

Loop statements는 반복문을 사용하는 방법입니다.
반복문에는 for, while이 해당됩니다.

for의 경우

범위가 숫자일 경우엔, [반복문 선언 for a in range, 파라미터 선언 x] --> for a in range(x): 와 같이 출력됩니다.
만약 범위에 숫자를 넣는 것이 아닌, 리스트를 넣는다면 [반복문 선언 for a in, 파라미터 선언 x ] --> for a in x: 와 같이 출력됩니다.

while의 경우

[반복문 선언 while, 논리 식 선언 ] --> while(논리 식): 와 같이 출력됩니다.

(파라미터 선언은 Assertion Parameters를 참고해주세요.)


(논리 식 선언 방법은 Bool logic을 참고해주세요.)

IF statements 사용 방법

IF statements에는 조건식을 사용하는 방법이 들어있습니다.
조건식에는 if, elif, else가 포함되며 이를 사용하시려면

if의 경우

[조건문 사용 if, 논리 식 선언 ] --> if(논리 식 선언): 와 같이 출력됩니다.

elif의 경우

[조건문 사용 else if, 논리 식 선언 ] --> elif(논리 식 선언): 와 같이 출력됩니다.

else의 경우

[조건문 사용 else, 논리 식 선언 ] --> else(논리 식 선언): 와 같이 출력됩니다.

(논리 식 선언 방법은 Bool logic을 참고해주세요.)


User Define Func Usage 사용 방법

User Define Func Usage에는 사용자가 함수를 정의하는 방법에 대해 설명되어 있습니다.
원하시는 함수명을 입력하시면 해당 함수가 정의됩니다.
예를 들자면, [함수 선언 aaa, 파라미터 선언 c] --> def aaa(c): 와 같이 출력됩니다.

(파라미터 선언 방법은 Assertion Parameters를 참고해주세요.)


Bool logic 사용 방법

Bool logic에는 논리 식을 사용하는 방법에 대해 설명되어 있습니다.
총 5가지의 논리식 선언이 가능하며, 가능한 논리식은 다음과 같습니다.

1: A는 B 와 같다

--> [논리식 선언 A는 B와 같다] --> A = B

2: A는 B 보다 크다

--> [논리식 선언 A는 B보다 크다] --> A > B

3: A는 B 보다 크거나 같다

--> [논리식 선언 A는 B보다 크거나 같다] --> A >= B

4: A는 B 보다 작거나 같다

--> [논리식 선언 A는 B보다 작거나 같다] --> A <= B

5: A는 B 보다 작다

--> [논리식 선언 A는 B보다 작다] --> A < B
