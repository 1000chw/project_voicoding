안녕하세요 저희는 경희대학교 컴퓨터공학과 19학번 민병수, 천현우 입니다.

저희는 프로젝트 Voicoding를 만드는 데에 {오픈소스 SW 개발} 과목에서 배운 Nodejs. Js. Git. 을 사용했으며, 과목에서 배우지 않고 사용한 언어는 Python, Html, CSS 등이 있습니다.
 
Voicoding을 사용하시려면, 위 메뉴바에서 Home을 클릭해주세요. (처음 메뉴 화면으로 돌아가고 싶으시다면 왼쪽 상단 위에 Voicoding을 눌러주시면 됩니다.) 
그리고, 메뉴바에 있는 Manual을 눌러 주시고, Manual에 맞춰서 입력을 하시려면 Speech 버튼을, 입력을 마치셨으면 To text버튼을 눌러 녹음을 종료해주세요. 

Transcription Raw Data에 말씀하신 내용이 나타나고 그 아래 TTP에 Python 코드로 변환된 내용이 삽입된 것을 확인하실 수 있습니다. 만약 의도한대로 입력이 되지 않았다면, 해당 코드를 클릭해 수정해주세요. 코드를 전부 입력하셨다면, 위 Run 버튼을 눌러주시면 코드가 작동하는 것을 확인할 수 있습니다. 작성하신 파일을 다운로드 받고 싶으시다면 TTP 레이아웃 왼쪽 상단 위에 존재하는 File을 누르시고 download를 누르시면 코드를 다운로드 받으실 수 있습니다. 

Voicoding의 모든 소스코드와 사용된 API, 그리고 참고 목록들은 Git repository, (http://khuhub.khu.ac.kr/2019102168/Voicoding) 에 모두 정리되어 공개되어 있습니다. 
문의사항은 메뉴바 오른쪽 상단에 있는 깃허브와 인스타를 통해 문의 바랍니다. 감사합니다.

Voicoding은 오픈소스로서 모두가 자유롭게 변형 및 사용이 가능합니다. 가능하시다면 출처 남겨주세요!
